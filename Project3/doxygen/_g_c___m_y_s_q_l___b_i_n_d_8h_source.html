10 /*
11  MYSQL_BIND is used by this Objective-C mysql library to pass data to
12  mysql_stmt_bind_param() and ultimately mysql_stmt_execute().
13 
14  The problem is that some members of the structure end up with pointers to auto collected
15  memory (in particular the results of calling -[NSString UTF8String]).
16 
17  Since these structure members don't have a __strong attribute, the memory can be garbage
18  collected and reused between the time of binding, and the call to mysql_stmt_execute().
19  This can result in garbage in the database.
20 
21  Here we redeclare this structure with the attributes, and then use this modified version.
22 
23  */
