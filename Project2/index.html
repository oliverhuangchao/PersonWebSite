682 Project Introductions---Book Cycle

In this project I plan to design a tool to provide a platform for stduents to exchange their books.

I plan to explore an App which can help students borrow book among themselves. As we know, books sometimes are really expensive for some of the students. However, most of the books are used only one semester. Normally, you have two choices: sell it or keep in your book store. Or you can choose to buy the e-book which is easy to store but have the similar price.

Now I can provide the third way: borrow it to someone strange and earn money from it.

The function includes:
  • Each user can sign up, login in search the current books available and see their details. The detail includes: book name, book owner, book author, return date and any other information (finished).
  • User can also see their borrow history (on going).
  • Book owners can see their books status (on going).
  • User can give comment and stars to all the books (on going).
  • Personal information protected (on going)
